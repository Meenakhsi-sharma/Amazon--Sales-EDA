# Amazon Sales Data Analysis(EDA)

 ## About Projects
 
This project focuses on performing Exploratory Data Analysis (EDA) on Amazon sales data to uncover trends, patterns, and business insights.

## Tools Used 

 * Python
* Pandas 
* Matplotlib / Seaborn 

## Key insights

- The "Set" category generated the highest revenue
- Peak sales observed in april 
-A declining trends after April indicates possible seasonality
- Western Dress showed consistent performance


## Conclusion 

This analysis helps in understanding customers behavior and suppoets better business decision making using data.
